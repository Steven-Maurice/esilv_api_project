Nous avons choisi d'utiliser arXiv pour récupérer les différents articles pour composer notre API, avec une grande et fiable
base de donnée. Cette dernière contient en effet beaucoup d'articles relatant des news ou des reflexions sur l'IA, ce qui
constitute le motif de notre recherche.

description des différents endpoints : 
index : bienvenu
/get_data : les liens pour accéder aux 5 derniers articles puliés sur arXiv sur le thème de l'IA et leurs catégories
/articles : Les informations de ces articles : id, lien, auteur(s), titre, date de publi, lien pdf, résumé, sous_catégorie, commentaires associés
/article/<id> : Permet d'obtenir le contenu d'un artcile identifié par son ID
/ml/sentiment/<id> : Renvoie une analyse de sentiment, si l'artcile est positif/négatif vis à vis du sujet de conversation et s'il est objectif ou subjectif  
/ml/keywords/<id> : Renvoie les Mots uqi apparaissent le plus fréquemment dans l'artcile avec leur nombre d'occurence
/ml/wordcloud/<id> : Renvoie un bel affichage de ces mots clés
/ml/compare/<id1>/<id2> : Renvoie la similarité de contenu entre les 2 articles 
/ml/resume/<id> : renvoie un résumé de l'article identifié par son id, avec la bibliothèque spacy
