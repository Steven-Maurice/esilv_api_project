API for ArXiv Scraping

Endpoints

/get_data/ - Default AI Topic

Fetches a list of articles related to AI. Returns the ID and title of up to 10 articles.

GET /get_data/
GET /get_data/{topic}

Replace {topic} with your search term to get related articles.

/articles - List of Articles

Displays a list of articles including arxiv_id, title, authors, published date, and link of up to 10 articles.

GET /articles

/article/{number} - Specific Article by ID

Accesses a specific article's content by ID and returns detailed information.

GET /article/{number}

Replace {number} with the article ID you want to retrieve.

/ml/{article_id} - Sentiment Analysis

Performs sentiment analysis on the abstract of the specified article and returns the result with an overall estimation.

GET /ml/{article_id}

Replace {article_id} with the article ID to analyze.

How to Use

To access the endpoints, you can use a browser or tools like curl or Postman. Simply append the endpoint to the base URL of this service and include the required parameters if needed. For instance, to get articles related to machine learning, you would access: 

GET /get_data/machine learning

To get sentiment analysis for a specific article, for example you would access: 

GET /ml/2404.03624v1
All endpoints return data in JSON format.
